## 🚀 Featured Work

### 🏥 Agentic Medical Prescription Reader
Digitization of handwritten prescriptions combining **multimodal vision** with a deterministic pipeline (clinical rules + domain validators). Diagnosis derivation via deterministic logic — independent of the LLM.  
`~90% precision` · `~USD 0.001 per prescription` · `Multimodal LLM` · `Computer Vision`

---

### 🤝 Multi-Agent ERP System (Natural Language)
Conversational multi-agent system to operate a professional practice in natural language.  
**Unified relational–vector–graph architecture** · 5-layer hierarchical memory · Tool Use via MCP · RAG · NLP-to-SQL · Gmail API integration.  
`Adoption in 1–2 weeks` · `Zero systems team intervention`

---

### 📈 Demand Forecasting with TFT
Seasonal staff demand model using **Temporal Fusion Transformers**.  
Optimized hiring planning and reduced emergency contracting dependency.  
`MAPE ~20%` · `PyTorch Forecasting` · `Time Series`

---

### 🏷️ Price Optimization Ensemble
Ensemble of **XGBoost · Random Forest · HistGradientBoosting · Ridge Regression** to predict optimal pricing strategies.  
`15% error` · `Outperformed human estimation` · `Increased quote success rate`

---

### 🛒 Retail Analytics — Computer Vision
Person detection system with **YOLO** deployed in production for two retail stores.  
Generates traffic metrics and wait times to optimize layout and operational efficiency.  
`Production deployment` · `Real-time inference` · `YOLO`

---

### 📦 Tire SKU Demand Forecasting
End-to-end forecasting for ~1,300 SKUs (1–6 month horizon) using an **LightGBM + CatBoost ensemble**, statistical fallback per horizon and **Conformal Prediction** for calibrated intervals.  
Deployed as an **async REST API**.  
`WAPE ~23% (Pareto-80 segment)` · `FastAPI` · `Conformal Prediction`
